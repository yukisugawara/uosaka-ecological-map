U-Osaka Ecological Map について

大阪大学（University of Osaka）に所属する研究者のネットワークをインタラクティブに可視化したマップです。

データソース

  • 大阪大学研究者総覧（rd.iai.osaka-u.ac.jp）-- 研究者リスト・所属・研究分野
  • researchmap（researchmap.jp）-- 論文・著書・研究発表・助成金の詳細情報

ネットワークの構築方法

ノード：各研究者を1つのノードとして表示。ノードの大きさはつながりの数（次数）に比例します。

エッジ（つながり）は以下の2種類：

  • 共著・共同研究関係（実線）-- 論文の共著者や科研費の共同研究者として名前が共に出現する回数をカウント
  • トピック類似度（破線）-- TF-IDFベクトルのコサイン類似度（閾値0.15以上）

色分けモード

  • トピックモード -- BERTopicにより研究内容を30のトピックに自動分類し色分け
  • 部局モード -- 所属部局（研究科・研究所等）ごとに色分け

つながり強度（%）

詳細パネルの「つながり%」は、共著件数×5 + トピック類似度のスコアを最大値で正規化した値です。

操作方法

  • 検索バーで研究者を名前・所属・分野で検索
  • ノードをクリックで選択 → 右パネルに詳細表示
  • 研究者選択後「経路検索」で2人の最短経路を表示
  • 凡例のトピック/部局をクリックでフィルタリング
  • マウスホイールでズーム、ドラッグで移動
  • 「保存」ボタンでネットワーク画像をPNGで保存
